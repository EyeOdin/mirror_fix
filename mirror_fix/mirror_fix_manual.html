Mirror Fix

Operators

Orientation : Selection the Orientation of the mirror operation to be applied to the opposite side.
Left : Left side is the Original Side and Direction is Horizontal.
Right : Right side is the Original Side and Direction is Horizontal.
Up : Up side is the Original Side and Direction is Vertical.
Down : Down side is the Original Side and Direction is Vertical.
Mirror : Mirror operation copy pastes with flip having sensibility to:
No Selection : When nothing is selected it will mirror the image considering the Original half side selected.
User Selection : When a user selection is present it will mirror only that given Selection to the other side.

This Script assumes the Mirror Axis is centered and when it is not it will be centered.
